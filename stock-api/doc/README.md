## 项目结构概览

```
stock-api/
├── job/              # 任务模块
│   └── scan/         # 股票筛选任务
├── strategy/         # 策略模块
│   └── conditions/   # 策略条件
├── data/             # 数据访问层
├── util/             # 工具模块
├── conf/             # 配置模块
└── doc/              # 文档目录（本目录）
```
